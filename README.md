# SQL to Excel Sales Dashboard Project

## Project Description
This project demonstrates how sales data can be extracted using SQL and transformed into a visual Excel dashboard. The purpose is to analyze sales performance over time and across products to support business decision-making.

## Tools Used
- MySQL 8.0 Command Line Client
- Microsoft Excel

## Database Structure
Table: sales  
Columns:
- order_id
- order_date
- product
- quantity
- price

## SQL Skills Demonstrated
- CREATE DATABASE and TABLE
- INSERT statements
- Aggregate functions (SUM)
- GROUP BY
- ORDER BY
- Date functions (MONTH)

## Excel Skills Demonstrated
- Data preparation
- Manual data structuring
- Chart creation (Column charts)
- Dashboard layout and formatting

## Key Insights
- Monthly sales trends were identified.
- Total sales per product were analyzed.
- Best-selling products were determined based on quantity sold.

## Outcome
The final Excel dashboard provides a clear visual summary of sales performance and demonstrates the ability to combine SQL data extraction with Excel-based reporting, suitable for entry-level data analysis and freelancing work.
